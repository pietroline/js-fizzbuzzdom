FizzBuzzDOM

In base al numero di box specificato dall'utente:
  • Se la posizione del box è multiplo di 3:
    • Il box conterrà la scritta Fizz
    • Il box avra il colore
  • Se la posizione del box è multiplo di 5:
    • Il box conterrà la scritta Buzz
    • Il box avra il colore
  • Se la posizione del box è multiplo di 3 e di 5:
    • Il box conterrà la scritta FizzBuzz
    • Il box avra il colore
  • In tutti gli altri casi, il box avrà colore e conterrà il numero di posizione
    • Se la posizione è pari, il numero sarà di colore bianco
    • Se la posizione è dispari, il numero sarà di colore nero
